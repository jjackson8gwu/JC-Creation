📊 Google Sheets Sync

⚙ One-Time Setup

  1. Open your Google Sheet, then click Extensions → Apps Script
  2. Delete any existing code, paste the script below, then save (Ctrl+S)
  3. In Apps Script, click the ⚙ Project Settings (gear icon, left sidebar)
  4. Scroll to Script Properties → Add script property and add all four:
    • GITHUB_TOKEN → your GitHub personal access token (needs repo scope)
    • GITHUB_OWNER → jjackson8gwu
    • GITHUB_REPO → JC-Creation
    • GITHUB_FILE → resources/products.json
  5. Click Deploy → New deployment → Type: Web app
    Set Execute as: Me  |  Who has access: Anyone
    Click Deploy, authorize when prompted, then copy the Web App URL
  6. Paste the URL in the box below and click Connect
  7. Back in Apps Script editor, run the createTrigger() function once — this enables automatic hourly sync from your sheet to the website
✅ How automatic sync works after setup:
• Customer buys online → website and Google Sheet quantities both decrease automatically
• You update quantities in your Google Sheet → website updates within the hour (via hourly trigger)
